041    /**
042     * This class provides an implementation of an LDAP connection pool, which is a
043     * structure that can hold multiple connections established to a given server
044     * that can be reused for multiple operations rather than creating and
045     * destroying connections for each operation.  This connection pool
046     * implementation provides traditional methods for checking out and releasing
047     * connections, but it also provides wrapper methods that make it easy to
048     * perform operations using pooled connections without the need to explicitly
049     * check out or release the connections.
050     * <BR><BR>
051     * Note that both the {@code LDAPConnectionPool} class and the
052     * {@link LDAPConnection} class implement the {@link LDAPInterface} interface.
053     * This is a common interface that defines a number of common methods for
054     * processing LDAP requests.  This means that in many cases, an application can
055     * use an object of type {@link LDAPInterface} rather than
056     * {@link LDAPConnection}, which makes it possible to work with either a single
057     * standalone connection or with a connection pool.
058     * <BR><BR>
059     * <H2>Creating a Connection Pool</H2>
060     * An LDAP connection pool can be created from either a single
061     * {@link LDAPConnection} (for which an appropriate number of copies will be
062     * created to fill out the pool) or using a {@link ServerSet} to create
063     * connections that may span multiple servers.  For example:
064     * <BR><BR>
065     * <PRE>
066     *   // Create a new LDAP connection pool with ten connections established and
067     *   // authenticated to the same server:
068     *   LDAPConnection connection = new LDAPConnection(address, port);
069     *   BindResult bindResult = connection.bind(bindDN, password);
070     *   LDAPConnectionPool connectionPool = new LDAPConnectionPool(connection, 10);
071     *
072     *   // Create a new LDAP connection pool with 10 connections spanning multiple
073     *   // servers using a server set.
074     *   RoundRobinServerSet serverSet = new RoundRobinServerSet(addresses, ports);
075     *   SimpleBindRequest bindRequest = new SimpleBindRequest(bindDN, password);
076     *   LDAPConnectionPool connectionPool =
077     *        new LDAPConnectionPool(serverSet, bindRequest, 10);
078     * </PRE>
079     * Note that in some cases, such as when using StartTLS, it may be necessary to
080     * perform some additional processing when a new connection is created for use
081     * in the connection pool.  In this case, a {@link PostConnectProcessor} should
082     * be provided to accomplish this.  See the documentation for the
083     * {@link StartTLSPostConnectProcessor} class for an example that demonstrates
084     * its use for creating a connection pool with connections secured using
085     * StartTLS.
086     * <BR><BR>
087     * <H2>Processing Operations with a Connection Pool</H2>
088     * If a single operation is to be processed using a connection from the
089     * connection pool, then it can be used without the need to check out or release
090     * a connection or perform any validity checking on the connection.  This can
091     * be accomplished via the {@link LDAPInterface} interface that allows a
092     * connection pool to be treated like a single connection.  For example, to
093     * perform a search using a pooled connection:
094     * <PRE>
095     *   SearchResult searchResult =
096     *        connectionPool.search("dc=example,dc=com", SearchScope.SUB,
097     *                              "(uid=john.doe)");
098     * </PRE>
099     * If an application needs to process multiple operations using a single
100     * connection, then it may be beneficial to obtain a connection from the pool
101     * to use for processing those operations and then return it back to the pool
102     * when it is no longer needed.  This can be done using the
103     * {@link #getConnection} and {@link #releaseConnection} methods.  If during
104     * processing it is determined that the connection is no longer valid, then the
105     * connection should be released back to the pool using the
106     * {@link #releaseDefunctConnection} method, which will ensure that the
107     * connection is closed and a new connection will be established to take its
108     * place in the pool.
109     * <BR><BR>
110     * Note that it is also possible to process multiple operations on a single
111     * connection using the {@link #processRequests} method.  This may be useful if
112     * a fixed set of operations should be processed over the same connection and
113     * none of the subsequent requests depend upon the results of the earlier
114     * operations.
115     * <BR><BR>
116     * Connection pools should generally not be used when performing operations that
117     * may change the state of the underlying connections.  This is particularly
118     * true for bind operations and the StartTLS extended operation, but it may
119     * apply to other types of operations as well.
120     * <BR><BR>
121     * Performing a bind operation using a connection from the pool will invalidate
122     * any previous authentication on that connection, and if that connection is
123     * released back to the pool then subsequent operation attempts may fail or
124     * be processed in an incorrect manner.  Bind operations should only be
125     * performed in a connection pool if the pool is to be used exclusively for
126     * processing binds, or if the bind request is specially crafted so that it will
127     * not change the identity of the associated connection (e.g., by including the
128     * retain identity request control in the bind request).
129     * <BR><BR>
130     * The StartTLS extended operation should never be invoked on a connection which
131     * is part of a connection pool.  It is acceptable for the pool to maintain
132     * connections which have been configured with StartTLS security prior to being
133     * added to the pool (via the use of the {@link StartTLSPostConnectProcessor}).
134     */

286      /**
287       * Creates a new LDAP connection pool with the specified number of
288       * connections, created as clones of the provided connection.
289       *
290       * @param  connection            The connection to use to provide the template
291       *                               for the other connections to be created.
292       *                               This connection will be included in the pool.
293       *                               It must not be {@code null}, and it must be
294       *                               established to the target server.  It does
295       *                               not necessarily need to be authenticated if
296       *                               all connections in the pool are to be
297       *                               unauthenticated.
298       * @param  initialConnections    The number of connections to initially
299       *                               establish when the pool is created.  It must
300       *                               be greater than or equal to one.
301       * @param  maxConnections        The maximum number of connections that should
302       *                               be maintained in the pool.  It must be
303       *                               greater than or equal to the initial number
304       *                               of connections.
305       * @param  postConnectProcessor  A processor that should be used to perform
306       *                               any post-connect processing for connections
307       *                               in this pool.  It may be {@code null} if no
308       *                               special processing is needed.  Note that this
309       *                               processing will not be invoked on the
310       *                               provided connection that will be used as the
311       *                               first connection in the pool.
312       *
313       * @throws  LDAPException  If the provided connection cannot be used to
314       *                         initialize the pool, or if a problem occurs while
315       *                         attempting to establish any of the connections.  If
316       *                         this is thrown, then all connections associated
317       *                         with the pool (including the one provided as an
318       *                         argument) will be closed.
319       */
320      public LDAPConnectionPool(final LDAPConnection connection,
321                                final int initialConnections,
322                                final int maxConnections,
323                                final PostConnectProcessor postConnectProcessor)
324             throws LDAPException
325      {
326        ensureNotNull(connection);
327        ensureTrue(initialConnections >= 1,
328                   "LDAPConnectionPool.initialConnections must be at least 1.");
329        ensureTrue(maxConnections >= initialConnections,
330                   "LDAPConnectionPool.initialConnections must not be greater " +
331                        "than maxConnections.");
332    
333        this.postConnectProcessor = postConnectProcessor;
334    
335        healthCheck         = new LDAPConnectionPoolHealthCheck();
336        healthCheckInterval = DEFAULT_HEALTH_CHECK_INTERVAL;
337        poolStatistics      = new LDAPConnectionPoolStatistics(this);
338        connectionPoolName  = null;
339    
340        if (! connection.isConnected())
341        {
342          throw new LDAPException(ResultCode.PARAM_ERROR,
343                                  ERR_POOL_CONN_NOT_ESTABLISHED.get());
344        }
345    
346    
347        serverSet = new SingleServerSet(connection.getConnectedAddress(),
348                                        connection.getConnectedPort(),
349                                        connection.getLastUsedSocketFactory(),
350                                        connection.getConnectionOptions());
351        bindRequest = connection.getLastBindRequest();
352    
353        final ArrayList<LDAPConnection> connList =
354             new ArrayList<LDAPConnection>(initialConnections);
355        connection.setConnectionName(null);
356        connection.setConnectionPool(this);
357        connList.add(connection);
358        for (int i=1; i < initialConnections; i++)
359        {
360          try
361          {
362            connList.add(createConnection());
363          }
364          catch (LDAPException le)
365          {
366            debugException(le);
367            for (final LDAPConnection c : connList)
368            {
369              try
370              {
371                c.setDisconnectInfo(DisconnectType.POOL_CREATION_FAILURE, null, le);
372                c.terminate(null);
373              }
374              catch (Exception e)
375              {
376                debugException(e);
377              }
378            }
379    
380            throw le;
381          }
382        }
383    
384        numConnections = maxConnections;
385    
386        availableConnections =
387             new LinkedBlockingQueue<LDAPConnection>(numConnections);
388        availableConnections.addAll(connList);
389    
390        failedReplaceCount        =
391             new AtomicInteger(maxConnections - initialConnections);
392        createIfNecessary         = true;
393        maxConnectionAge          = 0L;
394        minDisconnectInterval     = 0L;
395        lastExpiredDisconnectTime = 0L;
396        maxWaitTime               = 5000L;
397        closed                    = false;
398    
399        healthCheckThread = new LDAPConnectionPoolHealthCheckThread(this);
400        healthCheckThread.start();
401      }

576      /**
577       * Creates a new LDAP connection for use in this pool.
578       *
579       * @return  A new connection created for use in this pool.
580       *
581       * @throws  LDAPException  If a problem occurs while attempting to establish
582       *                         the connection.  If a connection had been created,
583       *                         it will be closed.
584       */
585      private LDAPConnection createConnection()
586              throws LDAPException
587      {
588        final LDAPConnection c = serverSet.getConnection(healthCheck);
589        c.setConnectionPool(this);
590    
591        // Auto-reconnect must be disabled for pooled connections, so turn it off
592        // if the associated connection options have it enabled for some reason.
593        LDAPConnectionOptions opts = c.getConnectionOptions();
594        if (opts.autoReconnect())
595        {
596          opts = opts.duplicate();
597          opts.setAutoReconnect(false);
598          c.setConnectionOptions(opts);
599        }
600    
601        if (postConnectProcessor != null)
602        {
603          try
604          {
605            postConnectProcessor.processPreAuthenticatedConnection(c);
606          }
607          catch (Exception e)
608          {
609            debugException(e);
610    
611            try
612            {
613              poolStatistics.incrementNumFailedConnectionAttempts();
614              c.setDisconnectInfo(DisconnectType.POOL_CREATION_FAILURE, null, e);
615              c.terminate(null);
616            }
617            catch (Exception e2)
618            {
619              debugException(e2);
620            }
621    
622            if (e instanceof LDAPException)
623            {
624              throw ((LDAPException) e);
625            }
626            else
627            {
628              throw new LDAPException(ResultCode.CONNECT_ERROR,
629                   ERR_POOL_POST_CONNECT_ERROR.get(getExceptionMessage(e)), e);
630            }
631          }
632        }
633    
634        try
635        {
636          if (bindRequest != null)
637          {
638            c.bind(bindRequest.duplicate());
639          }
640        }
641        catch (Exception e)
642        {
643          debugException(e);
644          try
645          {
646            poolStatistics.incrementNumFailedConnectionAttempts();
647            c.setDisconnectInfo(DisconnectType.BIND_FAILED, null, e);
648            c.terminate(null);
649          }
650          catch (Exception e2)
651          {
652            debugException(e2);
653          }
654    
655          if (e instanceof LDAPException)
656          {
657            throw ((LDAPException) e);
658          }
659          else
660          {
661            throw new LDAPException(ResultCode.CONNECT_ERROR,
662                 ERR_POOL_CONNECT_ERROR.get(getExceptionMessage(e)), e);
663          }
664        }
665    
666        if (postConnectProcessor != null)
667        {
668          try
669          {
670            postConnectProcessor.processPostAuthenticatedConnection(c);
671          }
672          catch (Exception e)
673          {
674            debugException(e);
675            try
676            {
677              poolStatistics.incrementNumFailedConnectionAttempts();
678              c.setDisconnectInfo(DisconnectType.POOL_CREATION_FAILURE, null, e);
679              c.terminate(null);
680            }
681            catch (Exception e2)
682            {
683              debugException(e2);
684            }
685    
686            if (e instanceof LDAPException)
687            {
688              throw ((LDAPException) e);
689            }
690            else
691            {
692              throw new LDAPException(ResultCode.CONNECT_ERROR,
693                   ERR_POOL_POST_CONNECT_ERROR.get(getExceptionMessage(e)), e);
694            }
695          }
696        }
697    
698        c.setConnectionPoolName(connectionPoolName);
699        poolStatistics.incrementNumSuccessfulConnectionAttempts();
700        return c;
701      }

706       * {@inheritDoc}
707       */
708      @Override()
709      public void close()
710      {
711        closed = true;
712        healthCheckThread.stopRunning();
713    
714        while (true)
715        {
716          final LDAPConnection conn = availableConnections.poll();
717          if (conn == null)
718          {
719            return;
720          }
721          else
722          {
723            poolStatistics.incrementNumConnectionsClosedUnneeded();
724            conn.setDisconnectInfo(DisconnectType.POOL_CLOSED, null, null);
725            conn.terminate(null);
726          }
727        }
728      }

743      /**
744       * {@inheritDoc}
745       */
746      @Override()
747      public LDAPConnection getConnection()
748             throws LDAPException
749      {
750        if (closed)
751        {
752          poolStatistics.incrementNumFailedCheckouts();
753          throw new LDAPException(ResultCode.CONNECT_ERROR,
754                                  ERR_POOL_CLOSED.get());
755        }
756    
757        LDAPConnection conn = availableConnections.poll();
758        if (conn != null)
759        {
760          if (conn.isConnected())
761          {
762            try
763            {
764              healthCheck.ensureConnectionValidForCheckout(conn);
765              poolStatistics.incrementNumSuccessfulCheckoutsWithoutWaiting();
766              return conn;
767            }
768            catch (LDAPException le)
769            {
770              debugException(le);
771            }
772          }
773    
774          handleDefunctConnection(conn);
775          for (int i=0; i < numConnections; i++)
776          {
777            conn = availableConnections.poll();
778            if (conn == null)
779            {
780              break;
781            }
782            else if (conn.isConnected())
783            {
784              try
785              {
786                healthCheck.ensureConnectionValidForCheckout(conn);
787                poolStatistics.incrementNumSuccessfulCheckoutsWithoutWaiting();
788                return conn;
789              }
790              catch (LDAPException le)
791              {
792                debugException(le);
793                handleDefunctConnection(conn);
794              }
795            }
796            else
797            {
798              handleDefunctConnection(conn);
799            }
800          }
801        }
802    
803        if (failedReplaceCount.get() > 0)
804        {
805          final int newReplaceCount = failedReplaceCount.getAndDecrement();
806          if (newReplaceCount > 0)
807          {
808            try
809            {
810              conn = createConnection();
811              poolStatistics.incrementNumSuccessfulCheckoutsNewConnection();
812              return conn;
813            }
814            catch (LDAPException le)
815            {
816              debugException(le);
817              failedReplaceCount.incrementAndGet();
818              poolStatistics.incrementNumFailedCheckouts();
819              throw le;
820            }
821          }
822          else
823          {
824            failedReplaceCount.incrementAndGet();
825            poolStatistics.incrementNumFailedCheckouts();
826            throw new LDAPException(ResultCode.CONNECT_ERROR,
827                                    ERR_POOL_NO_CONNECTIONS.get());
828          }
829        }
830    
831        if (maxWaitTime > 0)
832        {
833          try
834          {
835            conn = availableConnections.poll(maxWaitTime, TimeUnit.MILLISECONDS);
836            if (conn != null)
837            {
838              try
839              {
840                healthCheck.ensureConnectionValidForCheckout(conn);
841                poolStatistics.incrementNumSuccessfulCheckoutsAfterWaiting();
842                return conn;
843              }
844              catch (LDAPException le)
845              {
846                debugException(le);
847                handleDefunctConnection(conn);
848              }
849            }
850          }
851          catch (InterruptedException ie)
852          {
853            debugException(ie);
854          }
855        }
856    
857        if (createIfNecessary)
858        {
859          try
860          {
861            conn = createConnection();
862            poolStatistics.incrementNumSuccessfulCheckoutsNewConnection();
863            return conn;
864          }
865          catch (LDAPException le)
866          {
867            debugException(le);
868            poolStatistics.incrementNumFailedCheckouts();
869            throw le;
870          }
871        }
872        else
873        {
874          poolStatistics.incrementNumFailedCheckouts();
875          throw new LDAPException(ResultCode.CONNECT_ERROR,
876                                  ERR_POOL_NO_CONNECTIONS.get());
877        }
878      }

882      /**
883       * {@inheritDoc}
884       */
885      @Override()
886      public void releaseConnection(final LDAPConnection connection)
887      {
888        if (connection == null)
889        {
890          return;
891        }
892    
893        connection.setConnectionPoolName(connectionPoolName);
894        if (connectionIsExpired(connection))
895        {
896          try
897          {
898            final LDAPConnection newConnection = createConnection();
899            if (availableConnections.offer(newConnection))
900            {
901              connection.setDisconnectInfo(DisconnectType.POOLED_CONNECTION_EXPIRED,
902                   null, null);
903              connection.terminate(null);
904              poolStatistics.incrementNumConnectionsClosedExpired();
905              lastExpiredDisconnectTime = System.currentTimeMillis();
906              return;
907            }
908            else
909            {
910              newConnection.setDisconnectInfo(
911                   DisconnectType.POOLED_CONNECTION_UNNEEDED, null, null);
912              newConnection.terminate(null);
913              poolStatistics.incrementNumConnectionsClosedUnneeded();
914            }
915          }
916          catch (final LDAPException le)
917          {
918            debugException(le);
919          }
920        }
921    
922        try
923        {
924          healthCheck.ensureConnectionValidForRelease(connection);
925        }
926        catch (LDAPException le)
927        {
928          releaseDefunctConnection(connection);
929          return;
930        }
931    
932        if (availableConnections.offer(connection))
933        {
934          poolStatistics.incrementNumReleasedValid();
935        }
936        else
937        {
938          // This means that the connection pool is full, which can happen if the
939          // pool was empty when a request came in to retrieve a connection and
940          // createIfNecessary was true.  In this case, we'll just close the
941          // connection since we don't need it any more.
942          connection.setDisconnectInfo(DisconnectType.POOLED_CONNECTION_UNNEEDED,
943                                       null, null);
944          poolStatistics.incrementNumConnectionsClosedUnneeded();
945          connection.terminate(null);
946          return;
947        }
948    
949        if (closed)
950        {
951          close();
952        }
953      }

975      /**
976       * Performs the real work of terminating a defunct connection and replacing it
977       * with a new connection if possible.
978       *
979       * @param  connection  The defunct connection to be replaced.
980       *
981       * @return  The new connection created to take the place of the defunct
982       *          connection, or {@code null} if no new connection was created.
983       *          Note that if a connection is returned, it will have already been
984       *          made available and the caller must not rely on it being unused for
985       *          any other purpose.
986       */
987      private LDAPConnection handleDefunctConnection(
988                                  final LDAPConnection connection)
989      {
990        connection.setDisconnectInfo(DisconnectType.POOLED_CONNECTION_DEFUNCT, null,
991                                     null);
992        connection.terminate(null);
993    
994        if (closed)
995        {
996          return null;
997        }
998    
999        if (createIfNecessary && (availableConnections.remainingCapacity() <= 0))
1000        {
1001          return null;
1002        }
1003    
1004        try
1005        {
1006          final LDAPConnection conn = createConnection();
1007          if (! availableConnections.offer(conn))
1008          {
1009            conn.setDisconnectInfo(DisconnectType.POOLED_CONNECTION_UNNEEDED,
1010                                   null, null);
1011            conn.terminate(null);
1012            return null;
1013          }
1014    
1015          return conn;
1016        }
1017        catch (LDAPException le)
1018        {
1019          debugException(le);
1020          failedReplaceCount.incrementAndGet();
1021          return null;
1022        }
1023      }

1027      /**
1028       * Indicates whether the provided connection should be considered expired.
1029       *
1030       * @param  connection  The connection for which to make the determination.
1031       *
1032       * @return  {@code true} if the provided connection should be considered
1033       *          expired, or {@code false} if not.
1034       */
1035      private boolean connectionIsExpired(final LDAPConnection connection)
1036      {
1037        // If connection expiration is not enabled, then there is nothing to do.
1038        if (maxConnectionAge <= 0L)
1039        {
1040          return false;
1041        }
1042    
1043        // If there is a minimum disconnect interval, then make sure that we have
1044        // not closed another expired connection too recently.
1045        final long currentTime = System.currentTimeMillis();
1046        if ((currentTime - lastExpiredDisconnectTime) < minDisconnectInterval)
1047        {
1048          return false;
1049        }
1050    
1051        // Get the age of the connection and see if it is expired.
1052        final long connectionAge = currentTime - connection.getConnectTime();
1053        return (connectionAge > maxConnectionAge);
1054      }

1294      /**
1295       * {@inheritDoc}
1296       */
1297      @Override()
1298      protected void doHealthCheck()
1299      {
1300        // Create a set used to hold connections that we've already examined.  If we
1301        // encounter the same connection twice, then we know that we don't need to
1302        // do any more work.
1303        final HashSet<LDAPConnection> examinedConnections =
1304             new HashSet<LDAPConnection>(numConnections);
1305    
1306        for (int i=0; i < numConnections; i++)
1307        {
1308          LDAPConnection conn = availableConnections.poll();
1309          if (conn == null)
1310          {
1311            break;
1312          }
1313          else if (examinedConnections.contains(conn))
1314          {
1315            if (! availableConnections.offer(conn))
1316            {
1317              conn.setDisconnectInfo(DisconnectType.POOLED_CONNECTION_UNNEEDED,
1318                                     null, null);
1319              poolStatistics.incrementNumConnectionsClosedUnneeded();
1320              conn.terminate(null);
1321            }
1322            break;
1323          }
1324    
1325          if (! conn.isConnected())
1326          {
1327            conn = handleDefunctConnection(conn);
1328            if (conn != null)
1329            {
1330              examinedConnections.add(conn);
1331            }
1332          }
1333          else
1334          {
1335            if (connectionIsExpired(conn))
1336            {
1337              try
1338              {
1339                final LDAPConnection newConnection = createConnection();
1340                if (availableConnections.offer(newConnection))
1341                {
1342                  examinedConnections.add(newConnection);
1343                  conn.setDisconnectInfo(DisconnectType.POOLED_CONNECTION_EXPIRED,
1344                       null, null);
1345                  conn.terminate(null);
1346                  poolStatistics.incrementNumConnectionsClosedExpired();
1347                  lastExpiredDisconnectTime = System.currentTimeMillis();
1348                  continue;
1349                }
1350                else
1351                {
1352                  newConnection.setDisconnectInfo(
1353                       DisconnectType.POOLED_CONNECTION_UNNEEDED, null, null);
1354                  newConnection.terminate(null);
1355                  poolStatistics.incrementNumConnectionsClosedUnneeded();
1356                }
1357              }
1358              catch (final LDAPException le)
1359              {
1360                debugException(le);
1361              }
1362            }
1363    
1364            try
1365            {
1366              healthCheck.ensureConnectionValidForContinuedUse(conn);
1367              if (availableConnections.offer(conn))
1368              {
1369                examinedConnections.add(conn);
1370              }
1371              else
1372              {
1373                conn.setDisconnectInfo(DisconnectType.POOLED_CONNECTION_UNNEEDED,
1374                                       null, null);
1375                poolStatistics.incrementNumConnectionsClosedUnneeded();
1376                conn.terminate(null);
1377              }
1378            }
1379            catch (Exception e)
1380            {
1381              debugException(e);
1382              conn = handleDefunctConnection(conn);
1383              if (conn != null)
1384              {
1385                examinedConnections.add(conn);
1386              }
1387            }
1388          }
1389        }
1390      }
