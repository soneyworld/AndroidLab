This package contains the free edition of the UnboundID LDAP SDK for Java.  It
contains the following elements:

- unboundid-ldapsdk-se.jar -- The UnboundID LDAP SDK for Java (Standard
  Edition) library.

- docs/index.html -- A list of available documentation.

- examples -- A directory containing source code for example programs that
  illustrate the use of the LDAP SDK.

- LICENSE.txt -- Provides information about the licenses under which the
  UnboundID LDAP SDK for Java (Standard Edition) is made available.

- LICENSE-GPLv2.txt -- The license text for the GPLv2 license under which the
  UnboundID LDAP SDK for Java (Standard Edition) may be used.

- LICENSE-LGPLv2.1.txt -- The license text for the LGPLv2.1 license under which
  the UnboundID LDAP SDK for Java (Standard Edition) may be used.

- LICENSE-UnboundID-LDAPSDK.txt -- The license text for the UnboundID-specific
  free right to use license under which the UnboundID LDAP SDK for Java
  (Standard Edition) may be used.
